git merge 和 git rebase的差别

这时有人在origin中做了一些修改并提交，推送到origin了


这时你可以”pull”命令把”origin”分支上的修改拉下来并且和你的修改合并； 结果看起来就像一个新的”合并的提交”，合并时可以用rebase和merge

这时Git 会用两个分支的末端（C4 和 C6）以及它们的共同祖先（C2）进行一次简单的三方合并计算
